CARIOCA URBANAS - WEB RÁPIDA

QUÉ INCLUYE
- index.html
- style.css
- app.js
- carpeta img con las 23 páginas JPG exportadas de Canva

CÓMO PROBAR
1. Descomprimir este ZIP.
2. Abrir index.html en Chrome.

CÓMO CAMBIAR WHATSAPP
1. Abrir app.js.
2. Buscar WHATSAPP_NUMBER.
3. Reemplazar 543425980000 por el número real con código país, sin espacios ni guiones.

CÓMO ACTUALIZAR EL CATÁLOGO
1. Exportar las páginas desde Canva como JPG.
2. Reemplazar los archivos dentro de /img manteniendo nombres 1.jpg, 2.jpg, 3.jpg...
3. Si cambió la cantidad de páginas, editar TOTAL_PAGES en app.js.

PUBLICACIÓN GRATIS RECOMENDADA
- GitHub Pages o Cloudflare Pages.
- No depende de Canva como web.
- Canva queda como herramienta de diseño.
